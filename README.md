# PDF OCR Web Application

This project is a Python-based web application that allows users to upload PDF documents and extract text using Optical Character Recognition (OCR).

## Project Goals
- Upload PDF files via a web interface
- Extract text using Tesseract OCR
- Apply image preprocessing to improve OCR accuracy

## Tech Stack
- Python
- HTML
- CSS
- Flask
- OpenCV
- Tesseract OCR
- pytesseract

## Team Collaboration
This project is developed collaboratively as part of an internship Task.

## Setup
Create venv and install dependencies from requirements.txt.

python -m venv venv
source venv/bin/activate / venv\scripts\activate, On Windows

# install dependencies
pip install -r requirements.txt

# run flask server
python app.py | python ./app/app.py
